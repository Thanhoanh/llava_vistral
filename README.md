# Vistral Multimodal

## Features:
- Image understanding (Image → Text)
- Vietnamese medical image captioning
- Image generation (Text → Image) with StableDiffusion

## Usage on RunPod:
```bash
git clone[ https://github.com/Thanhoanh/llava_vistral.git
cd llava_vistral
pip install -r requirements.txt
python app.py
```
